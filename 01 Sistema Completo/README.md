Leer en este orden:

1. `indice-tematico.md` — Índice temático con referencias canónicas §X a cada concepto del framework.
2. `Bloque 1 — Fundamentos` — Intake Form, Fórmula maestra, Matriz base.
3. `Bloque 2 — Arquitectura` — URL Rules, Page Type Rules, Estructura de contenido, Internal Linking.
4. `Bloque 3 — Priorizacion` — Priority Score, QA Checklist, Producción en fases.
5. `Bloque 4 — Automatizacion` — Pseudocódigo, Master Prompt, Sistema final operativo.
6. `Bloque 5 — GBP` — GBP Creation & Website Alignment.

Principios del sistema:

- Web primero, GBP después.
- Main City = Madrid (ancla geográfica única para la fórmula).
- Local Coverage Areas no generan URLs por defecto (solo enriquecen contenido).
- Approved Expansion Areas generan URLs solo si se aprueban explícitamente.
- 28 páginas SEO base: `1 + S + 1 + S + A + G×S`.
- GBP Status = Not Created hasta que se ejecuta el Paso 14.

Cada subcarpeta de paso contiene:

- El archivo del sistema con índice canónico (§X en cada encabezado).
- El archivo de decisiones V1 (inventario de decisiones tomadas en ese paso).
