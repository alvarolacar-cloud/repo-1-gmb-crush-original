# Capa de Salida al Mercado — Módulos 15–19

Versión añadida al sistema GMB Crush existente.  
No sustituye los 14 pasos actuales.  
No modifica la arquitectura SEO, las reglas de URL, las decisiones/fuentes ni el Paso 14 de GBP.

---

## Objetivo de esta carpeta

Los pasos 1–14 responden a:

```text
qué páginas hay que crear
por qué existen
qué URL tienen
qué tipo de página son
cómo se conectan
qué fuente respalda cada decisión
cómo se alinea la web con el futuro GBP
```

Esta carpeta añade la capa que permite pasar de arquitectura SEO a web publicable:

```text
redacción final
diseño
construcción
QA de lanzamiento
escalado post-lanzamiento
```

---

## Módulos incluidos

| Módulo | Documento | Objetivo |
|---|---|---|
| 15 | `15-modulo-redaccion-final-por-url.md` | Convertir la arquitectura en contenido final por URL. |
| 16 | `16-modulo-extraccion-diseno.md` | Extraer identidad visual, tokens y layout-map. |
| 17 | `17-modulo-construccion-web.md` | Construir la web usando contenido + diseño + matriz de URLs. |
| 18 | `18-modulo-qa-lanzamiento.md` | Auditar técnica, contenido, diseño, SEO, schema y rastros IA antes de publicar. |
| 19 | `19-modulo-escalado-post-lanzamiento.md` | Medir, optimizar y decidir futuras expansiones sin romper la base. |

---

## Regla de integración

Estos módulos se ejecutan **después** de tener cerrados los pasos 1–14.

```text
Pasos 1–14 = sistema SEO y entidad local
Módulos 15–19 = producción y salida al mercado
```

---

## Orden de ejecución

```text
1. Ejecutar módulos 1–14 del sistema principal.
2. Confirmar URL Matrix, Page Type Rules, Content Architecture, Internal Linking, Priority Score, Publishing Plan y QA base.
3. Ejecutar Módulo 15 — Redacción final por URL.
4. Ejecutar Módulo 16 — Extracción de diseño.
5. Ejecutar Módulo 17 — Construcción web.
6. Ejecutar Módulo 18 — QA de lanzamiento.
7. Publicar.
8. Ejecutar Módulo 19 — Escalado post-lanzamiento.
```

---

## Regla sobre competidores

La parte de competidores **no se ejecuta en esta carpeta todavía**.

Cuando haga falta, se añadirá un módulo separado de:

```text
Contraste con competidores
SERP validation
Local Pack analysis
Gap analysis
```

Hasta entonces, cualquier decisión que requiera datos de competidores debe quedar como:

```text
Pendiente de módulo de competidores
```
