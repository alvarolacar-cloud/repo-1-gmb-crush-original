# Módulo 15c — Content Pack completo

Este archivo complementa el Módulo 15.

## Archivo principal

- `15c-content-pack-completo-para-construccion.md`

## Uso

Este documento debe entregarse al Módulo 17 — Construcción web.

Sirve para que la IA constructora no invente:

```text
URLs
H1
metadata
secciones
FAQs
CTAs
internal links
schema
datos bloqueados
```

## Inventario

```text
28 URLs SEO base
+ /contacto/ como página auxiliar
```

## Importante

Este pack es una especificación construible por URL.  
Si se necesita copy largo de 800–1.500 palabras por página, debe expandirse desde estas secciones sin modificar las decisiones, URLs, enlaces ni datos bloqueados.
