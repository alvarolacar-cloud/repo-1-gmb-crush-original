# Trazabilidad de fuentes — Sistema GMB Crush

Este pack contiene un archivo `.md` por cada paso del sistema.

Formato:

```text
ID | Decisión tomada | Valor decidido | Fuente | Justificación | Estado
```

Fuentes usadas:

```text
GMB Crush
Input humano
Geografía
Datos de búsqueda
Competidores
IA sin respaldo
```

Estados usados:

```text
Validada
Parcialmente validada
Pendiente de validación
IA sin respaldo
```

Nota:
`GMB Crush` agrupa tanto la doctrina de los frameworks como las reglas operativas cerradas del sistema.
